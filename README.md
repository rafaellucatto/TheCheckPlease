This was the first iOS app I ever made when I was starting to learn Swift from scratch (beginning with Storyboard for example).<br>
This is a very simple app, but it is protocol-oriented and made with MVVM with coordinator.<br>
Within this code, you can also find some samples of unit testing and some tests of snapshot.<br>
Basically it helps those who want to calculate how much they need to pay for something before ordering it.<br>
At the same time, you could also use it the other way around if you'd like to calculate how much you and/or a subset of your friends should pay for something afterwards.<br>
